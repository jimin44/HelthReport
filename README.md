# 운동관리 및 단백질 섭취 보조 앱<br/><br/>
## 사용 기술 및 개발환경
- JAVA
- Android Studio
- Firebase
<br/>

## 기능
- 로그인 및 회원가입
- 매일 새로운 운동 루틴 제공
- 단백질 섭취 관리
- 몸무게, 운동량, 단백질 섭취 통계 기록
